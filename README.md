# Meta Winner Ads Scraper

## Minimal Setup
1) `npm install`
2) `npm run setup`
3) `git init` / `git add .` / `git commit -m "init"` / `git remote add origin <YOUR_REPO_URL>` / `git push -u origin main`
4) Verify the GitHub Actions run

## Notes
- Do not double-click `.js` files on Windows. Use the npm commands above.
- If Node fetch has DNS issues on Windows, set `FORCE_IPV4=1` or `NODE_OPTIONS=--dns-result-order=ipv4first`.
